File Size Modified Actions

Restore a backup
Restore a .ndjson.gz file. The drop option is recommended.
Restore can overwrite data. Make a backup first if needed.
Tip
Use drop unless you know exactly what you're doing (otherwise risk of duplicates/errors).
How it works
  • Backup: creates a .ndjson.gz file (MongoDB export) and keeps it on the server.
  • Download: retrieves the file locally (useful before a risky operation).
  • Restore: imports the file. With drop, restored collections are deleted first.
Quick check
After restore, reload the page and check "Films", "Checklist" and "Stats".
